# maestro_detalle
## Repositorio para guardar los proyectos de diferente tecnologia o aquitectura para ejecutar un maestro detalle

### Variantes/opciones/versiones
- Windows Forms
	- Con ADO.Net (Usando procedimientos almacenados)
	- COn Entity Framework

- ASP.NET
	- Con MVC (Usando Entity Framework)
	- Con Web Forms (Usando ADO.NET con procedimientos almacenados)

### Tematicas
- Ahorros y aportes
- Creditos y pagos
- Ventas (Encabezado de factura y detalle de factura)
